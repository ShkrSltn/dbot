# DigiBot

DigiBot is an intelligent application built with Streamlit that leverages AI capabilities for various tasks.

## Features

- Profile Builder
- Enrichment Demo
- Batch Enrichment
- Quiz Interface

## Technology Stack

- **Frontend/Backend**: Streamlit
- **Database**: PostgreSQL
- **AI Integration**: LangChain, OpenAI
- **Data Processing**: Pandas, NumPy
